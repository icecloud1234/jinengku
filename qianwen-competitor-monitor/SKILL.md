**Requirements:** Python 3.8+, Chrome/Chromium, ChromeDriver (auto-installed), xvfb (Linux headless mode)

## Workflow

## 前置工作
文件就创建在工作目录即可，如果`logs/huawei`和`/logs/yingyongbao`文件存在的话就继续，如果不存在就创建

## 运行方式
在本地或远程主机上严格依次执行这三个脚本，并且只执行一次，不要使用其他方法。

不要修改任何脚本文件。不必理会脚本关于chromedriver的报错，那是正常的，不要尝试任何修改。

这个任务流程很长是正常的，不要自作主张。

## 第一步
```bash
python3 {baseDir}/scripts/huawei_scraper_selenium.py 
```

## 第二步
```bash
python3 {baseDir}/scripts/yingyongbao_scraper_selenium.py 
```

## 第三步
```bash
python3 {baseDir}/scripts/generate_summary.py
```

## 接下来

读取`summary_report.json`这个文件，然后生成一个Markdown的分析报告并保存这个Markdown的报告在桌面。

## `summary_report.json` 的格式

### JSON Summary (generated by Step 3)

`logs/summary_report.json` structure:
```json
{
  "generated_at": "2026-02-07T17:04:43.295337",
  "platforms": {
    "千问": {
      "huawei": {
        "download_count": "1.99亿",
        "rating": "3.3",
        "review_count": "1112",
        "url": "https://appgallery.huawei.com/app/C109126425"
      },
      "yingyongbao": {
        "download_count": "298.9万",
        "rating": "4.3",
        "review_count": "343"
      }
    }
  }
}
```
